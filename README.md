# eStocks

Aplicativo simples de controle de estoque desenvolvido em Flutter.

Criei esse projeto com um objetivo prático: ajudar uma ONG que atende pessoas em situação de vulnerabilidade.  
Eles precisavam organizar o pequeno almoxarifado e controlar melhor os itens recebidos e distribuídos.

Então decidi transformar isso em um projeto real de estudo e aplicação.

---

## O que o app faz

- Cadastro de itens com:
  - Nome
  - Quantidade
  - Categoria (Alimentos, Suplementos, Higiene)
  - Data de validade

- Organização por categorias

- Dashboard com visão geral:
  - Total de itens
  - Estoque baixo
  - Próximos do vencimento

- Alertas automáticos:
  - Notificação 7 dias antes do vencimento
  - Alerta quando o estoque está baixo

- Edição e exclusão de itens

- Modo escuro

---

## Tecnologias utilizadas

- Flutter
- Dart
- Hive (armazenamento local)
- Flutter Local Notifications
- UUID

---

## Motivação

Esse não é apenas um projeto de estudo.

Ele nasceu de uma necessidade real:  
organizar melhor os alimentos e itens básicos distribuídos por uma ONG que atua com pessoas em situação precária.

Além de praticar desenvolvimento mobile, busquei criar algo funcional, simples e realmente útil.

A ideia é continuar evoluindo o aplicativo conforme novas necessidades surgirem.
